# ArchVision: From CNNs to Multimodal

Large-scale scene classification project comparing CNNs, Vision Transformers, CLIP, ConvNeXt, Swin, EfficientNet, and LLaMA-Vision across 270+ experiments on 25,000+ images.

## Highlights

- 25,000+ images
- 270+ experiments
- CNNs, Transformers, CLIP, LLaMA-Vision
- Fine-tuning vs frozen model comparison
- Best performance: 95.6% with fine-tuned LLaMA-Vision
